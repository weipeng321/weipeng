git常用几个命令

November 14 2016

1.查看远程分支

git branch －a

2.查看本地分支

git branch

3.创建分支

git branch test

4.切换分支

git checkout test

5.删除本地分支

git branch －d test

6.删除远程分支

git push origin --delete <branch>

7.初始化仓库

git init

8.拉取远程仓库到本地

git clone [url]

9.关联远程仓库

git remote add [remote-name] [url]

10.添加所有文件

git add .

11.提交到本地仓库和add配合使用

git commit -m "注释"

12.撤销最近的一个提交

git revert HEAD

13.将本地代码提交到远程仓库

git push [remote-name] [loca-branch]:[remote-branch]

14.更新远程仓库

git pull [remote-name] [loca-branch]:[remote-branch]

15.合并分支

git merge [remote-name]/[branch]

16.查看当前本地仓库的所有分支

git branch

—

17.创建一个新分支

git branch (branch name)

18.删除一个分支

git branch -d(branch-name)

19.删除远程仓库的某个分支

git push (remote-name) :(remote-branch)

20.切换到某一个分支

git checkout [branch-name]

21.创建并切换分支

git checkout -b [branch-name]

22.查看操作日志

git log
